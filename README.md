# maze-solver
Visualization of different pathfinding algorithms.

## Installation
```bash
make
```
To remove:
```bash
make clean
```

## Usage
```bash
./solver [1-5] <maze.txt>
```
1. Random Search
2. DFS
3. BFS
4. Greedy Search
5. A*
